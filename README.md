# Fırat Otonom Kuraklık Motoru

Fırat Otonom Kuraklık Motoru; ArcGIS Pro tabanlı, Python ile geliştirilmiş bir CBS aracıdır. Landsat verilerini kullanarak NDWI ve NDVI indekslerini otonom hesaplar; bölgedeki kuraklık ve su varlığı değişimlerini manuel süreçleri ortadan kaldırarak hızlı, hatasız ve yüksek doğrulukla analiz eder. Akademik çalışmalar için uygundur.

## Proje İçeriği
Bu repo içerisinde, kuraklık analizini otonom hale getiren şu temel dosyalar bulunmaktadır:
- **Drought_Analysis_Toolkit.atbx**: ArcGIS Pro için geliştirilmiş araç kutusu (Toolbox).
- **Kod Dosyası**: Analiz süreçlerini yöneten Python betikleri.
- **Kılavuz (PDF)**: Aracın kurulumu ve kullanımı için hazırlanan teknik doküman.
- **Örnek Çıktı**: Analiz sonucunda üretilen örnek harita görseli.

## Nasıl Kullanılır?
1. Repoyu bilgisayarınıza indirin (Download ZIP).
2. `Drought_Analysis_Toolkit.atbx` dosyasını ArcGIS Pro üzerinden projenize ekleyin.
3. Analiz süreçlerini başlatmak için kılavuzda (PDF) belirtilen adımları izleyin.
